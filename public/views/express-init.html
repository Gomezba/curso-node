Primeros pasos con express

Introducción

En este capítulo vamos a construir un servidor web muy sencillo a través de Express para despues crear una API REST pero en varias secciones.

¿Qué es Express?

En la documentación oficial, Express nos dice, que es una infraestructura web, rápida, minimalista y flexible para NodeJs.

En términos generales para que sepamos exactamente qué es lo que hace Express, es interactuar con nuestro módulo HTTP.

Basicamente nos provee una serie de métodos y de propiedades que hace muy sencillo el interacturar con el módulo de HTTP.

Todo lo que hacemos con Express podemos hacerlo directamente con el módulo HTTP pero tendriamos que escribir mucho más código y seria más complejo de entender.

Sin embargo con Express crear un servidor web por ejemplo, es muchisimo mucho más sencillo y es lo que veremos en este capítulo para que podamos comparar que realemente veamos la facilidad que tiene utilizar Express para construir un servidor.

Pasos para empezar a construir nuestro servidor

Primero tenemos que saber que Express no es un módulo que venga por defecto, así que tenemos que instalarlo.

En nuestra terminal escribiremos npm install express -E al porner el prefijo -E signifca que instalará la version actual de Express y por lo tanto no agregará los caret en nuestro package-json.

Nosotros vamos a utilizar Express cuando el cliente interactue con nuestro servidor, así que por eso es una dependencia de producción y no de desarrollo.

Una vez que hacemos esto, ahora en la constante app tenemos guardado todo el objeto de Express con sus propiedades y sus métodos.
