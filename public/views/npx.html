Comando npx

El comando npx es una herramienta que viene con Node.js y se utiliza para ejecutar paquetes Node.js directamente desde el registro npm (Node Package Manager). La principal ventaja de npx es que permite ejecutar paquetes sin necesidad de instalarlos globalmente en nuestro sistema. Esto es útil cuando solo necesitamos ejecutar un comando específico de un paquete en lugar de tener que instalarlo en todo el sistema.

La función principal de npx es buscar el paquete en el registro npm y ejecutar el comando asociado a ese paquete. Si el paquete no está instalado localmente, npx lo instalará temporalmente para ejecutar el comando y luego lo eliminará.

La versión de Node.js en la que se introdujo npx es la 5.2.0. A partir de esa versión, npx se incluyó en la instalación predeterminada de Node.js.

El comando npx es útil porque podemos ejecutar paquetes que anteriormente teniamos que instalar de forma global pero sin la necesidad de instalarlos directamente en nuestro sistema operativo y que permanezcan a parte en el registro de nuestro proyecto.

Esto logra que tengamos mas limpio nuestro equipo de computo y de nuestro proyecto de Nodejs y sus dependencias.

Hay que saber que cuando nosotros instalamos un paquete de forma global, este no se instala como dependencia en nuestro package.json por lo cual no se ve reflejado en la carpeta node_modules si no que se instala en nuestro equipo de computo en la carpeta donde nosotros tenemos instalado NodeJs y ahi se encuentra una carpeta de node_modules pero de la instalación de node.

Es por eso que el comando npx nos ayuda a tener más limpio nuestro equipo y las dependencias de cada proyecto.

Lista de librerias y herramientas que suelen recomendar npx para su uso

  • Eleventy: Compila un proyecto Eleventy, un generador de sitios estáticos (SSG).

    npx eleventy

  • Astro: Crea un nuevo proyecto Astro, un marco de trabajo para construir aplicaciones web rápidas utilizando JavaScript moderno.

    npx create-astro

  • Vue CLI: Similar a create-react-app, pero para proyectos Vue.js.

    npx vue create my-vue-app

  • Angular CLI: Inicia un nuevo proyecto Angular sin necesidad de instalar globalmente Angular CLI.

    npx @angular/cli new my-angular-app

  • Gatsby: Inicia un nuevo sitio Gatsby, una aplicación de generación de sitios estáticos basada en React.

    npx gatsby new my-gatsby-site

  • TypeScript: Ejecuta el compilador TypeScript sin tener que instalarlo globalmente.

    npx tsc

  • http-server: Inicia un servidor HTTP para servir archivos estáticos desde el directorio actual.

    npx http-server

  • storybook: Inicia un entorno de desarrollo para componentes de UI con Storybook.

    npx -p @storybook/cli sb init

  • jest: Ejecuta pruebas con Jest, un popular marco de pruebas para JavaScript.

    npx jest
